# Contributing to Prosthetic Access Atlas

This is an open-access resource. We welcome contributions from researchers, clinicians, policymakers, and advocates.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add your contribution'`)
4. Open a Pull Request

## What to Contribute

- Updated clinical trial data from ClinicalTrials.gov
- New underserved region gap analyses
- Additional provider proximity data
- Policy or Medicaid expansion updates
- Local O&P (Certified Prosthetist/Orthotist) provider listings
- Geographic data (shapefiles, GeoJSON)

## Data Sources

- ClinicalTrials.gov API
- OpenStreetMap (geocoding, POI search)
- U.S. Census Bureau (Medicaid expansion, amputation rates)
