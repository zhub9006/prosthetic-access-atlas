# Prosthetic Access Atlas — Findings Update

Consolidated findings compiled July 21, 2026.

## Clinical Trial Trends
- 644 total prosthetic studies on ClinicalTrials.gov
- 117 limb-amputation focused studies
- Only ~20% actively recruiting
- Phase 3 trials critically scarce (~30 globally)
- Academic sponsors dominate (76.6%)

## Gap Analysis — 3 Underserved Regions
- Rural West Virginia (Charleston/Beckley)
- Eastern Kentucky (Hazard/Pikeville)
- Mississippi Delta (Greenville/Clarksdale)

### Finding: Zero CPO providers within 60 km of any region
- Nearest CPO: 130–200+ miles away
- WV and MS have not expanded Medicaid
- MS Delta has the highest amputation rate in the US

## Methodology
- ClinicalTrials.gov API v2
- OpenStreetMap geocoding + provider search
- 50-60 km search radius around target cities
