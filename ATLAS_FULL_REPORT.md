## 2. Trial Trend Analysis

### Temporal Trends

- **2018–2020**: Pre-pandemic; steady enrollment in osteoarthritis and joint prosthesis studies
- **2021–2022**: COVID-19 disruption; many deferred/resumed studies; emerging focus on infection risk
- **2023–2024**: Recovery period; new device trials emerging (microprocessor knees, osseointegration)
- **2025–2026**: Active recruitment for next-gen prosthetics; Gaza conflict creating new prosthetic need studies

- **Upper Limb Prosthetics**: ~15%

## 7. Data Sources & Methodology

### Clinical Trials Data
- **Source**: ClinicalTrials.gov API (clinicaltrials.gov/api/v2/studies)
- **Search Terms**: "prosthetic", "amputation", "prosthetic limb", "prosthetic rehabilitation"
- **Total Studies Indexed**: 644
- **Data Retrieved**: Status, phase, sponsor, location, intervention, outcomes, eligibility
- **Limitations**: API timeout issues encountered with large queries; analysis based on available subset

### Gap Analysis Data
- **Source**: OpenStreetMap (via Overpass API / OSM MCP Server)
- **Search Method**: Neighborhood analysis + route directions for three rural regions
- **Search Radius**: 100 km from each center point
- **Provider Categories Searched**: Prosthetic/Orthotic care, rehabilitation centers, hospitals with prosthetics departments
- **Results**: 0 providers found in all three regions

### Infrastructure Scores
- **Source**: OpenStreetMap neighborhood analysis
- **Categories**: Groceries, Restaurants, Healthcare, Education, Public Transport, Parks, Sports, Entertainment, Shopping, Services
- **Score Range**: 0–10 (higher = more features within radius)

### Route Data
- **Source**: OSM Routing API (car mode)
- **Destinations**: Charleston, WV (for Beckley & Pikeville); Memphis, TN (for Greenville)
